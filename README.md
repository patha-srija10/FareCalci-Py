FareCalc - Travel Fare Optimizer

A console-based Python application that calculates ride fares for a ride-sharing service based on distance, vehicle type, and surge pricing.

Features
- Dynamic fare calculation using dictionary mapping
- Surge pricing during peak hours (5 PM – 8 PM)
- Error handling for invalid vehicle types
- Simple console-based receipt generation

Technologies Used
- Python

How to Run
1. Clone the repository
2. Open the project in VS Code
3. Run:
   python farecalc.py

Example
Enter distance in km: 10  
Enter vehicle type: Economy  
Enter hour: 18  

Output:  
Total Fare: ₹150.0
